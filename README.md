<h4 align='center'>Configuration for neovim, zsh, fzf, iterm and more.</h4>

### Homebrew

Install `brew` packages from Brewfile

```sh
$ make brew
```

### kitty

> Installed by brew

All config located and loaded by `kitty/kitty.conf`
